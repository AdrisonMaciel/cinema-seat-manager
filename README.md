# 🎬 Cinema Seat Manager

Um sistema simples de gerenciamento de assentos de cinema com funcionalidades de reserva, cancelamento, visualização do mapa e controle de arrecadação, implementado em Java.

---

## ✨ Funcionalidades

- Visualizar mapa de assentos (ocupados/livres)
- Reservar assento (inteira ou meia)
- Cancelar reserva
- Visualizar total de assentos e valor arrecadado
- Persistência dos dados em arquivo 

---

## 🛠️ Tecnologias

- Java (JDK 8+)
- Terminal/console
- Serialização de objetos 

---

📷 Exemplo de Execução

1. Mostrar mapa de assentos
2. Reservar assento
3. Cancelar reserva
4. Mostrar total de assentos e arrecadação
5. Sair
